# PalindroneCheckerApp

// running procedure
1. Start the application using an IDE.
2. Take input from the user.
3. Then we either convert the whole string into uppercase or to lowercase to avoid disparancies.
4. Reverse the string and compare it with the original string
5. If it is same output : palindrone if not output : Not a Palindrone.

// Objective

Palindorne Checker App is used to compare strings or numbers. Palindrone means the string or the number
need to be same whether we read it from backward order or forward order this is considered to be a
palindrone number or string. For example : radar, 1221......
